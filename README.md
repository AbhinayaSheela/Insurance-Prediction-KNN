#  Insurance Cost Prediction using KNN Regression

##  Project Overview
This project uses a **K-Nearest Neighbors (KNN) Regression model** to predict medical insurance charges based on user details such as age, BMI, smoking status, and more. The model is deployed using **Streamlit** for an interactive web application.


##  Dataset Features

- age  
- sex  
- bmi  
- children  
- smoker  
- region  
- charges (Target)


##  Target Variable
- **charges** → Predicted insurance cost (continuous value)


##  Technologies Used

- Python  
- Pandas  
- NumPy  
- Scikit-learn  
- KNN Regression  
- Streamlit  
- Matplotlib  
- Seaborn  


##  Machine Learning Model

- Algorithm: K-Nearest Neighbors Regressor  
- Scaling: StandardScaler  
- Problem Type: Regression
